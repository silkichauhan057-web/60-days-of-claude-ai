# Day 7 — My Claude Usage Strategy

## Task
Used the "Claude Usage Strategy" prompt on Claude to get a personalized recommendation for which model (Haiku/Sonnet/Opus) and effort level to use for my daily tasks.

## My Profile
- **Situation:** Student
- **Usage Frequency:** Daily
- **Primary Activities:** Learning, Coding, Business Planning
- **Output Needs:** Fast Answers, Learning Support, Coding Help, Deep Research, Business Strategy, Creative Work

## Recommended Primary Model
**Claude Sonnet 5** — best balance of speed and reasoning for daily learning + coding + business planning work.

- **Haiku 4.5:** Quick facts, formatting, simple rewrites/summaries
- **Sonnet 5:** Daily coding help, learning explanations, first-pass business drafts (my default, ~70-80% of tasks)
- **Opus 4.8:** Deep research, complex business strategy, hard coding/architecture problems

## Effort Levels
- **Low:** Fast lookups, simple rewrites
- **Standard (default):** Coding help, explanations, first drafts
- **High:** Step-by-step reasoning, debugging subtle bugs, comparing strategies
- **Max:** Full business plans, deep multi-source research, architecture decisions

## Workflow Table

| Task | Best Model | Best Effort | Reason |
|---|---|---|---|
| Daily concept learning | Sonnet 5 | Standard | Clear explanations, no overkill |
| Quick fact-check | Haiku 4.5 | Low | Speed matters, low complexity |
| Debugging code | Sonnet 5 | Standard → High | Escalate if stuck |
| New coding project | Sonnet 5 | High | Needs planning across files |
| Business plan draft | Sonnet 5 | Standard | Good structure, fast iteration |
| Business strategy | Opus 4.8 | High/Max | High-stakes reasoning |
| Deep research | Opus 4.8 | Max | Needs synthesis |
| Creative writing | Sonnet 5 | Standard | Strong quality, fast |
| Study notes/summaries | Haiku 4.5 / Sonnet 5 | Low/Standard | Simple transformation |

## Final Recommendation
**Claude Sonnet 5, Standard effort** — if I could use only one model+effort combo for most work, this covers learning, coding, and business planning well without the slowdown of higher tiers. Opus 4.8 reserved for genuinely hard/high-stakes problems.

## Key Learnings
- Matching model tier to task complexity saves time and improves output quality.
- Sonnet 5 is the right daily-driver for a student balancing multiple activities.
- Reserve Opus 4.8 for genuinely high-stakes or complex multi-step problems.
- Effort/depth level matters as much as model choice — escalate only when needed.
- A documented workflow table removes the "which model right now" decision fatigue.
